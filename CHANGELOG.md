# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-03

### Added
- Initial release
- Interactive Bash script for Minikube cluster management
- Multi-node Kubernetes cluster creation (3 nodes)
- Docker, kubectl, and Minikube installation
- Auto-start cluster after login
- Auto-stop cluster on logout/shutdown
- Desktop shortcut creation
- Safe cluster deletion
- Full system wipe option
- Ingress addon support

### Configuration
- Kubernetes version: v1.30.2
- Memory per node: 3072 MB
- CPUs per node: 2
- Disk size: 40 GB
- Container runtime: containerd
